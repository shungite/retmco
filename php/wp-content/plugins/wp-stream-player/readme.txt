== Description ==

<p class=\"MsoPlainText\">WP Stream Player</p> <p class=\"MsoPlainText\"></p> <p class=\"MsoPlainText\">WP Stream Player is an audio streaming player plugin for wordpress. It can plays variety of formats: MP3, OGG, nsv, RTP, ASX with fallback to flash and wmp. The shotrtcode generator make easy to use the stream player and include it anywhere in your blog/website.</p> <p class=\"MsoPlainText\">Options:</p> <p class=\"MsoPlainText\"> </p> <p class=\"MsoPlainText\">- Compatibility with MP3/OGG/RTMP/WEBM Streams</p> <p class=\"MsoPlainText\">- Compatibility with ASX Streams (WMP Plugin automatic integration)</p> <p class=\"MsoPlainText\">- Fallback from HTML5 to Flash for old browsers</p> <p class=\"MsoPlainText\">- Shortcode Generator from Settings</p> <p class=\"MsoPlainText\">- iOS/Android Compatibility</p>

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
